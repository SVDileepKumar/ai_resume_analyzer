# Security

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report details privately to the repository maintainers (use GitHub Security Advisories if enabled for this repo, or contact the owner via their profile email).

Include:

- A description of the issue and its impact
- Steps to reproduce (proof of concept if possible)
- Suggested fix or patch (optional)

We aim to acknowledge reports within a few business days.

## Scope

This application may process resume and job-description text. Deploy only over HTTPS, restrict access appropriately, and follow your organization’s data-handling policies.
